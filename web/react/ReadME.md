# React 项目创建

## Next.js

```
npx create-next-app@latest
```

## Remix



```
npx create-remix
```



## Expo（用于原生应用）

提供Native React https://reactnative.dev/ 

```
npx create-expo-app <项目名>
```



创建普通react应用

```
npx create-react-app <项目名>
```

创建带sw的项目

```
npx create-react-app <项目名> --template cra-template-pwa

```
